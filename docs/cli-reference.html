はじめに CLI Reference

CLI Reference

Hügeltonプロジェクトで使用するCLIコマンドのリファレンス。

▶ WebMCP で開く

arduino-cli (RP2040)

arduino-cli compile -b rp2040:rp2040:rpipico \
  --board-options="usbstack=tinyusb" \
  --output-dir build/ \
  --export-binaries \
  /path/to/sketch

Wokwi CLI

export WOKWI_CLI_TOKEN="your-token"
wokwi-cli /path/to/project --timeout 5000

ローカル UF2/ELF を直接指定する場合は wokwi.toml をプロジェクトルートに配置：

[wokwi]
version = 1
firmware = 'build/sketch.ino.uf2'
elf = 'build/sketch.ino.elf'
rfc2217ServerPort = 4000

picotool (RP2040)

コマンド用途
picotool load firmware.uf2UF2をPicoに書き込み
picotool info接続情報
picotool save -t elf firmware.elfFlashからELFに保存
