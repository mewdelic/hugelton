はじめに

Hügelton Instruments のドキュメントへようこそ。

Hügeltonは神戸を拠点にする電子楽器／モジュラーシンセを自作するスタジオです。オープンソース・DIY・カウンターカルチャーをモットーに、ハードウェアからソフトウェアまで一貫して設計しています。

▶ WebMCP で開く

プロジェクト一覧

現在開発中のプロジェクト：

  • K112E — Dual VCF / State Variable Filter（開発中）
  • Polume — 12鍵 Capacitive Touch Keyboard / CV Interface（開発中）
  • Clockworker — 96 PPQN Eurorack Clock/DIN Sync Module（試作中）
  • Quex — LuaJIT Rhythm Composer（開発中）
  • CrispyZebra (PicoCZ) — CZ-101 Phase Distortion on RP2040（試作中）

Hügeltonの哲学

すべてのプロジェクトは以下の原則に基づいて設計されています：

OSS  —  すべてをオープンソースで
DIY  —  自分で作れるものを
RETRO  —  レトロな美学と感触を
COUNTER-CULTURE  —  主流に抗う独自の道を
