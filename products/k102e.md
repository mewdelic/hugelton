K102E is a digital compound oscillator for Eurorack synthesizers based on unique "Island Synthesis" concept. Three independent oscillators and diverse wavetable synthesis realize rich sound generation.

## Features

- Three independent oscillators for rich sound generation
- DSP processing via microcontroller provides stable oscillation and diverse wavetable synthesis
- Main oscillators (A/B) function similar to general compound oscillators
- Three synthesis modes: Phase Modulation, Multiplication, Crossfade
- Hard sync and detune functions
- 8 waveform types inspired by 80s wavetable synthesizers
- Harmonic-specific index control for wavetable position
- Third oscillator (C) functions as auxiliary oscillator
- Selectable frequency range from low to high
- Can also be used as LFO or cross-modulator
- External control interface including standard CV input (octave/volt control), harmonics CV (wavetable position), and alpha parameter CV for waveform synthesis

## Specifications

- **Size**: 20HP (101.6mm) x 3U (128.5mm) x 20mm Eurorack
- **Power**: +12V: 300mA / -12V: 10mA
- **CV Inputs**: V/Oct, Harmonics, Alpha CV (0-5V)
- **Audio**: 3.5mm mono jack (±5Vpp)
- **Oscillators**: 3x Independent (A/B/C)
- **Wavetables**: 8x Wavetable Types
- **Modes**: Phase Mod / Multiplication / Crossfade
