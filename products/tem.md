Converts environmental sensor data to control voltage and gate signals. A networked environmental modulator that brings the outside world into your modular synthesizer.

## Features

- Environmental sensor input (light, temperature, humidity)
- CV output for each parameter
- Gate output for threshold detection
- Network connectivity capabilities
